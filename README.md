# BbsSample
BbsSample ; React + SpringBoot

## 기능
### Bbs
- BBS CRUD 
list(+paging), write, detail, update, delete / +답글(부모글 삭제할 경우 답글도 안보이게 처리함)
- BBS 댓글 기능
- BBS 조회수 증가 기능

<br/>

### Login
- 간단한 로그인, 로그아웃
- 회원가입
- 로그인시 id 기억하기 기능 (cookie 이용)
